# StartUpWorld

A web-based hybrid terminal + management sim built with React, TypeScript, Zustand, and TailwindCSS.

## Local setup

```bash
npm install
npm run dev
```

## Structure

```
src/
  engine/        # pure simulation logic
  ui/
    components/
    panels/
    terminal/
  state/
  types/
  styles/
```

## Deployment

This is a static Vite app. Deploy directly to Vercel.
